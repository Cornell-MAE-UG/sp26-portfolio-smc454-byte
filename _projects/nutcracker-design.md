**Find**
The goal of this project is to design a mechanical nutcracker that can efficiently crack a nut using a system of levers. The objective is to determine the mechanical advantage and ensure that the applied hand force produces a sufficiently large compressive force on the nut.

**Given**
Hand force = 200 N
Distance from pivot to hand= 0.20 m
Distance from pivot to nut= 0.02 m
Required nut cracking force= 2000 N

**Approach**

**1. Model the nutcracker as a lever with the pivot as the fulcrum

2. Apply static equilibrium ΣM = 0

3. Use moment equations
M = F × d
F = applied hand force
d = perpendicular distance from pivot**

F_hand × L_hand = F_nut × L_nut

**4. Solve for output force**

F_nut = (F_hand × L_hand) / L_nut
F_nut = (200 × 0.20) / 0.02
F_nut = 40 / 0.02
F_nut = 2000 N

**5. Solve for output force**

F_nut = (F_hand × L_hand) / L_nut
F_nut = (200 × 0.20) / 0.02
F_nut = 40 / 0.02
F_nut = 2000 N

**Usability:**
The nutcraker I designed has a simple design with low user effort, but it could  be improved by creating longer handles to lower the hand force required to crack the nut.
